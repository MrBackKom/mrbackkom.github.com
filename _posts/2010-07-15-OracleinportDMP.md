菜单栏 --> Tools --> Export Tables... --> Oracle Export
 
Export Executable  :   D:\oracleclient\bin\EXP.EXE
Output File : C:\Documents and Settings\Administrator\桌面\eam.dmp
 
参照上面的截图设置正确后单击右下角的 Export按钮即可。
